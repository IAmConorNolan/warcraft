# Warcraft

Warcraft is a library written in Swift that can be used to supercharge your development time when writing applications for the game World of Warcraft.

## Installation

### Swift Package Manager

### Cocoapods

```ruby
pod 'Warcraft'
```

```bash
pod install
```

### Carthage


## Usage

```swift
import Warcraft
```
